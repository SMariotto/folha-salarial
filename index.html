Informações

Sobre a ferramenta

INSS progressivo

Calculado faixa a faixa conforme a tabela vigente. Cada faixa salarial é tributada pela alíquota correspondente, não pelo total.

Imposto de renda

A base de cálculo é o salário bruto menos INSS e deduções. O sistema usa automaticamente a dedução mais vantajosa: simplificada (R$ 607,20) ou legal (INSS + dependentes + pensão).

Redução do IR (Lei 15.270/2025)

Para base de cálculo até R$ 5.000, o imposto é zerado. Entre R$ 5.000,01 e R$ 7.350,00, a redução é aplicada de forma decrescente.
